This iframe contains a page element animated via CSS to change the height of the page. This is not a recommended use of this library, as it places a large load on the browser. If you really need to do this, then it is recommended that you use taggedElement as your heightCalculationMethod to minimise the overhead of working out the page height. See the source code of this page for an example of using this method.
